# Herm - The TUI for Hermes Agent

> **herm** /hɜːm/ _noun_
>
> A sculptured head of Hermes on a square stone pillar, used in ancient Greece as a boundary marker or signpost at crossroads.

## Prerequisites

- [Bun](https://bun.sh/) - JavaScript runtime and package manager
- [Hermes Agent](https://github.com/BionicLabsHQ/hermes-agent) - The AI agent backend
- Terminal with 256 color support
